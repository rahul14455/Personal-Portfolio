My Projects

EaseMyTrip Clone

I built a responsive clone of EaseMyTrip using React, demonstrating modern web development skills. The application features a user-friendly, responsive interface with multi-modal booking options, including flights, hotels, trains, buses, and holiday packages. I implemented user authentication, advanced search functionality, and simulated booking processes to enhance the user experience. React Router was used for seamless navigation, and the Context API was employed for efficient state management. Real-time travel data was integrated through external APIs, and performance was optimized through effective rendering strategies. The project was deployed on Netlify, reflecting end-to-end execution of a full-stack application.

Tech Stack: HTML, CSS, JavaScript, React

LinkedIn Clone
